# 🍗 Cozinheiras da Vila Site simples e moderno para divulgação e vendas de coxinhas artesanais. ## 📋 Sobre o Projeto Este projeto foi criado em HTML e CSS puro, funcionando perfeitamente no GitHub Pages sem necessidade de instalar React ou frameworks. O site possui: - Página inicial - Área de produtos - Informações de contato - Design responsivo - Layout moderno --- ## 🚀 Como Publicar no GitHub Pages 1. Crie um repositório no GitHub 2. Adicione o arquivo `index.html` 3. Faça o commit dos arquivos 4. Vá em: Settings → Pages → Deploy from Branch 5. Selecione: - Branch: `main` - Pasta: `/root` 6. Clique em **Save** Seu site ficará online em alguns segundos. --- ## 💻 Tecnologias Utilizadas - HTML5 - CSS3 --- ## 📷 Funcionalidades ✔️ Página inicial atrativa ✔️ Cards de produtos ✔️ Botões de ação ✔️ Layout responsivo ✔️ Rodapé com contato --- ## 📞 Contato 📍 Santo Antônio da Platina - PR 📱 (43) 99999-9999 --- ## © Direitos Projeto criado para divulgação de vendas de coxinhas artesanais.
